# 🌿 Nawa Cahaya — Sistem Dashboard

Platform donasi investasi transparan untuk yayasan pemberdayaan anak.

## Login Demo
- Admin: `admin` / `admin168`
- Donatur: `donatur` / `donor88`
- Relawan: `relawan` / `relawan88`
- Publik: Pilih tab Publik → langsung masuk

## Setup Google Sheets
1. Buat Google Sheets → Extensions → Apps Script
2. Paste `Code.gs` → tambah `form.html`
3. Deploy → Web App → Anyone
4. Paste URL di banner atas dashboard

## Tech Stack
- Vanilla HTML/CSS/JS (no framework, zero dependencies*)
- Chart.js (CDN)
- Google Apps Script (backend)
- Google Sheets (database)
