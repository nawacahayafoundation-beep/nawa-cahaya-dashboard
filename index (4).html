📋 Panduan Setup Google Sheets + Apps Script (5 Langkah)
1
Buka Google Sheets baru
Pergi ke sheets.google.com → buat spreadsheet baru → beri nama "Nawa Cahaya Data"
2
Buka Apps Script
Di Google Sheets → klik Extensions → Apps Script → tab baru terbuka
3
Paste Code.gs
Hapus semua kode default → paste isi file Code.gs yang sudah didownload → Ctrl+S (Save)
4
Deploy sebagai Web App
Klik Deploy → New Deployment → Web App → Execute as: Me → Who has access: Anyone → klik Deploy
5
Copy URL → Paste di atas → Klik Hubungkan
Copy URL deployment yang muncul (format: https://script.google.com/macros/s/xxx/exec) → paste di field di atas → klik Hubungkan & Test → banner hijau = berhasil! ✅
💡 Setelah terhubung → klik Sync di topbar → data dari Google Sheets langsung masuk dashboard. Setiap isi form publik juga langsung masuk ke Sheets.
